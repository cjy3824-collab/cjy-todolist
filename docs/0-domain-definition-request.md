너는 최고의 비즈니스 도메인 전문가야. "cjy-todoList"라는 이름의 사용자 인증과 함께 사용하는 TodoList 애플리케이션을 개발하려고 해. 다음 내용을 바탕으로 적절한 서브에이전트를 활용해 도메인 정의서 초안을 작성해줘.

# 문제 정의 :

- 사용자마다 할일 목록을 관리할 수 있는 앱이 있으면 좋겠다.
- 할일의 만료 기간이 표시되어 할일 목록을 관리할 수 있으면 좋겠다.
- 국경일과 같은 일정(할일)이 함께 관리될 수 있어야 한다.

# Team CalTalk 앱의 주요 기능 :

- 모든 사용자는 인증(로그인)후 애플리케이션의 기능을 이용한다.
- 로그인한 사용자는 자신이 등록한 할일 목록을 조회할 수 있다.
- 할일 목록에 새로운 할일을 추가할 수 있다.
- 기존에 등록된 할일을 수정할 수 있다.
- 기존에 등록되 할일을 삭제할 수 있다. 삭제된 할일은 휴지통으로 들어가고, 다시 복구할 수 있어야 한다.
- 휴지통에 있는 할일은 영구히 삭제할 수 있다.
- 국경일과 같은 공통의 할일은 관리자만 수정, 삭제할 수 있다.
- 캘린더가 표시되고, 각 할일의 기한을 캘린더에 표시해야 한다.
